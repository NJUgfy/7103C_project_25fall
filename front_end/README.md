# 前端项目运行文档
## 项目信息
- **技术栈**: Vue 3 + Vite + Element Plus
- **开发者**: 任苗苗
- **版本**: v1.0

---
## 环境要求
- **Node.js**: >= 20.19 或 22.12+
- **npm**: >= 10.x

---

## 部署
#### 1. 安装依赖
```bash
npm install
```

#### 2. 启动服务器

```bash
npm run dev
```

---

## 开发运行
- 修改后端接口地址：编辑 `src/utils/request.js` 中的 `baseURL` 配置
